# 📌 What is a Component in Angular?

A **Component** is the basic building block of any Angular application.

It controls:
- **What the user sees** (HTML)
- **How it looks** (CSS)
- **How it behaves** (TypeScript logic)

A component =  
**TypeScript (logic) + HTML (UI) + CSS (styles)**

Every component must have:
1. A **TypeScript class**
2. A **@Component decorator**
3. A **template** (HTML)
4. Optional **styles** (CSS)
5. A **selector** (used to place the component on the page)
