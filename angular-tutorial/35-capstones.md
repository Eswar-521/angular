# Capstone Projects

1) **Admin CRUD + Auth** — Reactive Forms, guards, Http interceptors.
2) **E-commerce Front** — Products, cart, checkout; NgRx for state; lazy routes.
3) **Realtime Chat** — WebSocket gateway, RxJS streams, presence.
4) **Analytics Dashboard** — Charts + CDK virtual scroll; server filtering.
5) **Universal Blog** — SSR, i18n, SEO; preloading and edge caching.
**Expected output:** A portfolio that demonstrates Angular strengths across forms, state, RxJS, and SSR.
